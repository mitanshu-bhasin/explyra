# Explyra Skills & Marketplace

Explyra Skills is a peer-to-peer marketplace and community built with React, TypeScript, and Vite. It allows users to share, discover, and trade skills within the Explyra ecosystem.

## Features
- **Skill Feed:** Discover what others are learning and sharing.
- **Marketplace:** Buy, sell, or trade professional services and skills.
- **User Profiles:** Showcase your skills, experience, and marketplace listings.
- **Social Integration:** Follow other users and get notified of their activities.

## Development

```bash
# Install dependencies
npm install

# Run the development server
npm run dev

# Build for production
npm run build
```

## Structure
- `/src/pages` - Contains the main views (SkillFeed, Marketplace, UserProfile, etc.)
- `/src/components` - Reusable UI components.
- `/src/context` - Global application state.
