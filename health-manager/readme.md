# HealthZen Pro - AI Powered Health Ecosystem 🧬

HealthZen Pro is an advanced **Progressive Web App (PWA)** designed to be your holistic health companion. It combines manual tracking with **Artificial Intelligence** to generate personalized workout and meal plans based on your specific biometrics (Body Fat, BMI, Health Conditions).

## 🚀 Live Demo
[https://healthzenmb.netlify.app](https://healthzenmb.netlify.app)

## ✨ Key Features
- **Advanced PWA:** Installable on iOS and Android with offline support.
- **AI Health Coach:** Generates custom JSON-based diet & workout plans using Llama 3 models.
- **Biometric Analysis:** Auto-calculates BMI and tracks Body Fat %.
- **Environmental Sensing:** Real-time AQI, UV Index, and Weather data for workout recommendations.
- **Calorie & Macro Tracker:** Visual database for Indian & International foods.
- **Intimacy & Wellness:** A hidden tracker for holistic well-being.
- **Medicine Cabinet:** Reminder system for daily medications.

## 🛠️ Tech Stack
- **Frontend:** HTML5, JavaScript (ES6+), Tailwind CSS
- **Database:** Google Firebase Firestore
- **Auth:** Firebase Authentication
- **AI:** Groq API (Llama-3.3-70b-versatile)
- **Charts:** Chart.js
- **Icons:** FontAwesome 6
