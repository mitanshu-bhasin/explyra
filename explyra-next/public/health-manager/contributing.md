# Contributing to HealthZen Pro 🤝

Thank you for your interest in contributing to HealthZen Pro! We welcome contributions from everyone.

## How to Contribute

1.  **Fork the repository** to your GitHub account.
2.  **Clone the project** locally:
    ```bash
    git clone [https://github.com/your-username/healthzen-pro.git](https://github.com/your-username/healthzen-pro.git)
    ```
3.  **Create a new branch** for your feature or bug fix:
    ```bash
    git checkout -b feature-name
    ```
4.  **Make your changes** and commit them.
5.  **Push to your branch** and submit a **Pull Request (PR)**.

## Reporting Bugs

If you find a bug, please open an issue on GitHub with:
- A description of the bug.
- Steps to reproduce it.
- Screenshots (if applicable).
